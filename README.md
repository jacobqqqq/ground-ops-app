# Ground Ops

Role-based airport ground operations app for shift visibility and handoff continuity.

**Live demo:** [warehouse-handoff-hub.lovable.app](https://warehouse-handoff-hub.lovable.app)

---

### Views

**Manager** — full dashboard with active crew, aircraft queue, turnaround metrics, SFO weather conditions, active incidents, shift log, and handoff acknowledgment.

**Ground Ops** — scoped view with current shift status, assigned gate, and active incidents only.

---

### Stack
React · TypeScript · Tailwind · shadcn/ui · Vite
